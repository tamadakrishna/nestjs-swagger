## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ git clone https://github.com/tamadakrishna/nestjs-swagger.git

$ cd nestjs-swagger

$ npm install

$ npm run start
```

## Swagger Installation

```bash
$ npm install --save @nestjs/swagger
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
